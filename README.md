# Bezel

---

Bezel is a modern **responsive landing page** built with **Bootstrap**, **Font Awesome**, and **Animate.css**. It’s designed to showcase a company’s services, team, pricing, and testimonials with smooth animations and a professional layout.  

---
## Features
- Fully **responsive** for desktop, tablet, and mobile  
- Hero **carousel** with call-to-action buttons  
- **Team section** with social media overlays  
- **Services** and **pricing** sections  
- **Testimonials** carousel  
- Contact form with **Google Maps** embed  
- Smooth **animations** using Animate.css  

---
## Technologies Used
- **Bootstrap 5**  
- **HTML5 & CSS3**  
- **Font Awesome**  
- **Animate.css**
